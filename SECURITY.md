# 安全说明

## 支持版本

在正式发布分支出现之前，安全修复默认只维护最新的 `main` 分支。

## 报告安全问题

如果发现安全问题，请优先通过 GitHub Security Advisory 私下报告；如果仓库后续提供了安全联系人，也可以按仓库说明联系维护者。

请不要在公开 Issue 中包含私有账号 Cookie、授权头、个人内容或第三方私有数据。

## 安全边界

本项目设计目标是：

- 只提取公开页面响应中已经存在的媒体元数据。
- 页面请求和重定向只允许小红书相关页面域名。
- 将 `/proxy` 限制在明确允许的小红书 HTTPS 媒体域名内。
- 不向上游媒体 CDN 转发网站 `Referer`。
- 支持通过 `ALLOWED_ORIGINS` 配置 CORS Origin 白名单。

本项目不能用于，也不应被修改为：

- 绕过登录、验证码、付费墙、风控或权限控制。
- 访问私有或未授权内容。
- 作为开放代理使用。
- 收集用户凭据、Cookie 或 Session Token。

欢迎反馈任何有助于维护这些安全边界的问题。
